# TennisClubSaaS.Frontend

Frontend estático en Vanilla JS para SaaS_Tenis_Club.

## Ejecutar local

Abrir `index.html` o servir esta carpeta:

```bash
npx serve .
```

## Configurar API

Editar `assets/js/config.js`:

```js
apiBaseUrl: "https://tu-api.somee.com"
```

## Publicar en Netlify

Usar como publish directory:

```text
frontend/TennisClubSaaS.Frontend
```

No hay build command.
